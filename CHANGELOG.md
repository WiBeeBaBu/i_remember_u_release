## v1.3.7 (2026-05-13)

[下载 I Remember U_1.3.7_x64-setup.exe](https://github.com/WiBeeBaBu/i_remember_u_release/releases/tag/v1.3.7)

**新增**
- ECDSA P-256 激活码系统（离线验证，一码绑定 3 台设备）
- Supabase 匿名统计（启动计数 Counter 模式 + 失败限 3 条/设备）
- UID 撤销列表（手动令单个激活码失效，无需轮换密钥）
- 启动日志上报激活码标识（license_uid 字段，方便定位用户）

**优化**
- 出生年份设为必填项（`*` 标注，未填红色提示 + 自动滚动定位）
- 姓名校验对齐（未填红色提示 + 滚动，与出生年份一致）
- 删除新建联系人页重复的性别下拉框
- 重置数据库改为二次确认（点击后需再点"确定"，防误触）
- 设置页卡片重新排序（生日提醒 → 自动主题 → 数据库管理 → 日历导出 → 关闭行为 → 激活码信息 → 关于）
- CustomSelect 组件支持 error 态（红色边框）
- Vite 5.4 → 6.4 + @vitejs/plugin-vue 5 → 6

**修复**
- 激活成功后卡死在"正在进入应用…"（路由守卫缓存缺陷）
- 开发模式下打开数据库/导出目录跳转到错误路径
- 数据目录统一到 `%LOCALAPPDATA%\IRememberU`（开发/发行一致）
- 关闭行为在发行版不生效（AppState 初始化遗漏）
- CSP 开放 Supabase 域名（发行版 telemetry 被拦截）
- 移除 `<input required>` 原生属性（浏览器拦截自定义校验）
