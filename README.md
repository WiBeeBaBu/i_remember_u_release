## 这是什么

**I Remember U** 帮你记录朋友和家人的一切：生日、喜好、性格、重要日期、礼物灵感、互动历史。在合适的时候提醒你，让你成为那个总是记得的人。

---

## 功能

- 生日仪表盘（公历农历双支持）
- 全方位人物画像（30+ 信息维度）
- 智能提醒（自定义时间 + 提前天数，毫秒级推送）
- 礼物管家（灵感 + 已送清单）
- 互动时间线（见面/通话/消息记录）
- 日历视图（节日 + 纪念日 + 自定义标注）
- 数据自主（本地存储，一键导出/导入 JSON）

## 常见问题

**Q: 下载的 exe 安装程序无法打开，双击无反应？**

A: 右键 exe 安装程序 → 进入文件属性 → 底部属性勾选"解除锁定" → 确定。Windows 对下载的可执行文件会自动锁定，解除后即可正常运行。
